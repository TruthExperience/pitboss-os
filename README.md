# PitBoss OS

A multi-tenant League Operating System powering TRL, WSC, SRH, and future leagues.

This monorepo includes:

- Next.js 14 PWA (apps/web)
- Discord Bot (apps/bot)
- Internal LLM Runtime + TeraBox Loader (apps/ai)
- Background Workers (apps/workers)
- Shared Domain Logic (packages/core)
- Shared Types (packages/types)
- Shared Utilities (packages/shared)
- Supabase Schema + Migrations (supabase)

Package manager: pnpm
